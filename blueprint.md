
# Lotto Number Generator

## Overview

A simple, modern web application to generate and display 5 sets of lottery numbers. The application will be user-friendly, visually appealing, and responsive, featuring a theme toggle for light and dark modes.

## Style, Design, and Features

### Implemented

*   **Initial Setup:**
    *   `index.html`: Main application structure.
    *   `style.css`: Styles for the application.
    *   `main.js`: Core logic for number generation.
    *   `blueprint.md`: Project documentation.
*   **Core Functionality:**
    *   Generates 5 unique sets of 6 lottery numbers (1-45).
    *   Displays the generated numbers in a clear, organized layout.
    *   Includes a button to trigger number generation.

### Design

*   **Theme:** Switchable dark and light themes.
    *   **Dark Mode:** A dark, gradient background with glowing elements for a futuristic feel.
    *   **Light Mode:** A clean, bright interface for high readability.
*   **Layout:** Centered content with a clear and intuitive layout.
*   **Typography:** Expressive fonts to emphasize key information.
*   **Visual Effects:** Multi-layered drop shadows for a sense of depth.
*   **Interactivity:** Interactive buttons with a "glow" effect in dark mode.

### Features

*   **Number Generation:** Generate 5 unique sets of 6 lottery numbers.
*   **Display:** Clearly display the 5 generated sets of numbers.
*   **"Generate" Button:** A button to generate new sets of numbers.
*   **Theme Toggle:** A button to switch between dark and light modes, with the preference saved for future visits.
